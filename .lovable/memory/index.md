# Project Memory

## Core
Elegant Beige/White/Black theme. Accents: Brown-Black (#2b2725) & Gold (#9b6b3f).
Typography: System font stack (-apple-system, SF Pro Display, Segoe UI). No external fonts. Swiss spelling ('ss', no 'ß').
Product: "RAJ NEXUS" (3-in-1 charger, Qi2.2, 25W, CHF 99.–). Space Black only.
Tech: Headless Shopify via Storefront API, Supabase (RLS), Framer Motion.
Routing: `/` is Launch Page. `/shop` or `?mode=shop` for shop view.
